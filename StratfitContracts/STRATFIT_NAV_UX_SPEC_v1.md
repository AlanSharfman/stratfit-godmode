---

STRATFIT NAVIGATION UX SPEC v1

PURPOSE  
Define the visual structure, layout, and behavioural expectations for the primary navigation to ensure clarity, hierarchy, and usability.

NAV PRINCIPLES  

Clear hierarchy  
Logical flow order  
Strong visual separation  
Immediate orientation  
Consistent typography  

---

NAV STRUCTURE  

Primary flow order:

Initiate  
Objectives  
Position  
Studio  
Compare  
Risk  
Valuation  
Assessment  
Roadmap  

---

VISUAL GROUPING  

GROUP 1 — SETUP  
Initiate  
Objectives  

GROUP 2 — UNDERSTAND  
Position  

GROUP 3 — EXPLORE  
Studio  
Compare  

GROUP 4 — ANALYZE  
Risk  
Valuation  

GROUP 5 — DECIDE  
Assessment  
Roadmap  

Each group must have visual separation.

---

LAYOUT RULES  

Navigation must include:

Section dividers  
Active state highlight  
Hover state feedback  
Clear spacing between groups  

No continuous list without separation.

---

TYPOGRAPHY  

Use one consistent font family.  
Clear weight hierarchy:

Primary nav items → medium/semibold  
Group labels → small caps or lighter weight  

Avoid overly small text or inconsistent sizing.

---

ACTIVE STATE  

Active page must be visually distinct using:

Accent color  
Indicator bar or highlight  
Bold or weight shift  

User must always know current location.

---

UX BEHAVIOUR  

Navigation must:

Remain persistent  
Provide immediate feedback  
Avoid clutter  
Avoid overcrowding  

---

SUCCESS CRITERIA  

User instantly understands:

Where they are  
Where they can go  
How sections relate
